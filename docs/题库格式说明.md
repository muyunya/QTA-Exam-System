# 题库JSON格式说明

本文档详细说明题库文件的JSON格式规范。

## 基本结构

```json
{
    "meta": {
        "name": "题库名称",
        "version": "1.0",
        "author": "作者名",
        "description": "题库描述"
    },
    "questions": [
        // 题目数组
    ]
}
```

## 题目通用字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | number | 是 | 题目唯一ID |
| type | string | 是 | 题目类型 |
| question | string | 是 | 题目内容 |
| difficulty | string | 否 | 难度：easy/medium/hard |
| tags | array | 否 | 标签数组 |
| explanation | string | 否 | 答案解析 |

## 各题型格式

### 1. 单选题 (single_choice)

```json
{
    "id": 1,
    "type": "single_choice",
    "difficulty": "easy",
    "tags": ["基础"],
    "question": "问题内容",
    "options": [
        "A. 选项1",
        "B. 选项2",
        "C. 选项3",
        "D. 选项4"
    ],
    "answer": "A",
    "explanation": "答案解析"
}
```

### 2. 多选题 (multi_choice)

```json
{
    "id": 2,
    "type": "multi_choice",
    "question": "问题内容（多选）",
    "options": [
        "A. 选项1",
        "B. 选项2",
        "C. 选项3",
        "D. 选项4"
    ],
    "answer": ["A", "B", "D"],
    "explanation": "答案解析"
}
```

### 3. 填空题 (fill_blank)

```json
{
    "id": 3,
    "type": "fill_blank",
    "question": "Python使用 ____ 关键字定义函数",
    "answer": ["def"],
    "explanation": "答案解析"
}
```

多个填空：
```json
{
    "question": "Python使用 ____ 定义函数，使用 ____ 定义类",
    "answer": ["def", "class"]
}
```

支持多个正确答案（用|分隔）：
```json
{
    "answer": ["def|DEF"]
}
```

### 4. 判断题 (judge)

```json
{
    "id": 4,
    "type": "judge",
    "question": "Python是一种编译型语言。",
    "answer": false,
    "explanation": "Python是解释型语言"
}
```

### 5. 简答题 (short_answer)

```json
{
    "id": 5,
    "type": "short_answer",
    "question": "请简述列表和元组的区别",
    "answer": "参考答案内容...",
    "keywords": ["可变", "不可变"],
    "explanation": "答案解析"
}
```

### 6. 编程题 (coding)

```json
{
    "id": 6,
    "type": "coding",
    "question": "编写一个函数计算两数之和",
    "code_template": "def add(a, b):\n    # 请在此处编写代码\n    pass",
    "test_cases": [
        {"input": "1, 2", "expected_output": "3"},
        {"input": "10, 20", "expected_output": "30"}
    ],
    "answer_code": "def add(a, b):\n    return a + b",
    "language": "python",
    "explanation": "答案解析"
}
```

## 注意事项

1. 所有JSON文件必须使用UTF-8编码
2. 题目ID在同一题库中必须唯一
3. 代码模板中的换行使用 `\n`
4. 测试用例的input和expected_output都是字符串格式
