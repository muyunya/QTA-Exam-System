# QTA 刷题软件 使用说明

## 🚀 快速开始

### 启动程序
```bash
cd d:\Script\QTA
python main.py
```

### 第一次使用
1. 启动程序后，点击左侧「📂 打开题库」按钮
2. 选择 `data/questions/sample.json` 示例题库
3. 选择练习模式开始刷题

---

## 📚 功能说明

### 1. 题目类型
| 图标 | 类型 | 说明 |
|------|------|------|
| 🔵 | 单选题 | 点击选项选择答案 |
| 🟣 | 多选题 | 勾选多个正确答案 |
| 🟢 | 填空题 | 在输入框填写答案 |
| 🟡 | 判断题 | 选择正确/错误 |
| 🩷 | 简答题 | 输入文字回答 |
| 🔷 | 编程题 | 编写代码并运行验证 |

### 2. 练习模式
- **顺序练习**：按题目顺序依次练习
- **随机练习**：打乱顺序随机练习
- **错题练习**：只练习之前做错的题
- **收藏练习**：只练习收藏的题目

### 3. 主要功能
- ⭐ **收藏题目**：点击右上角收藏按钮
- 📊 **学习统计**：侧边栏显示答题统计
- 📈 **详细统计**：点击「查看详细统计」按钮
- 🌙 **主题切换**：切换日间/夜间模式

## 📝 自制题库

题库使用JSON格式，放置在 `data/questions/` 目录下。

### 基本格式
```json
{
    "meta": {
        "name": "我的题库",
        "version": "1.0",
        "author": "作者",
        "description": "描述"
    },
    "questions": [
        {
            "id": 1,
            "type": "single_choice",
            "question": "问题内容",
            "options": ["A. 选项1", "B. 选项2"],
            "answer": "A",
            "explanation": "解析"
        }
    ]
}
```

### 题目类型值
- `single_choice` - 单选题
- `multi_choice` - 多选题
- `fill_blank` - 填空题
- `judge` - 判断题
- `short_answer` - 简答题
- `coding` - 编程题

详细格式参考：`docs/题库格式说明.md`

## ❓ 常见问题

**Q: 程序启动失败？**
> 确保已安装依赖：`pip install ttkbootstrap pygments`

**Q: 题库加载失败？**
> 检查JSON格式是否正确，确保使用UTF-8编码

**Q: 编程题代码运行报错？**
> 编程题仅支持Python代码，且有5秒执行时限
